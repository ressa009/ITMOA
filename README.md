## <b># 프로젝트 개요</b> 
학생은 필요한 정보를 찾을 수 있고 학원은 광고할 수 있는
'IT 학원 정보 공유 플랫폼'

<br>

---
## <b># 참여내용</b> 
#### <b>- 기여도: 35%</b>
#### <b>- 프로젝트 기획 : 프로젝트 기획문서 작성 주도 </b>
- 개발범위화면목록
- 플로우차트
- 와이어프레임
- 스토리보드
  
#### <b>- 프로젝트 구현</b> <br>
     1. <b>학원 검색 페이지</b>
        - 학원 전체 목록 제공
        - 학원 목록에 대한 지역/지점/과정별 조건 검색 <br><br>
     2. <b>학원 정보 상세 페이지</b>
        - 학원의 특정 클래스 상세 조회 및 출력 (기간별 수업내용 및 학원 위치)
        - 해당 클래스에 대한 상담예약 및 좋아요 기능 <br><br>
     3. <b>관리자 페이지 학원관리</b>
        - 학원 신규 등록 및 학원 정보 수정
        - 등록된 학원 목록 출력 및 조건별 검색<br><br>
     4. <b>전체 SQL문 작성</b>

<br><br><br>
